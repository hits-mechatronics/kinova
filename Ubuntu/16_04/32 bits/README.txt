To install the complete package, please execute the script installSDK32.sh

- The script will ask for root privileges to install all the components.
- The default installation folder is /opt/kinova
- At the root of the package, you will find 2 icon files: SDK.png (Development Center) and Icon_torque (Torque Console). they can be used for launchers and shortcuts.

If you encounter any problems, please contact support@kinovarobotics.com
